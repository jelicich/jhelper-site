What's jHelper about?

It is a Chrome extension that will make your everyday work easier helping you keep track of your Jira tickets by storing, copying and accessing them quicker.

Features

  • Add tickets

    Use the custom button from Jira to save tickets, so you can access them very quickly from the extension.

  • Suggested List

    A list with the latest tickets you visited or interacted with will show up so you can add them from there.

  • Search

    Find what you are looking for in both, your saved and suggested tickets.

  • Follow Sprint

    Add the sprints you are working on and see the main information in just one click.
