# jhelper-site
jHelper Chrome extension website

<h3>Installation</h3>

Clone the repo and run:<br />

npm run setup<br />

<h3>Development</h3>

Run the following command:<br />

npm run dev<br />

Open your browser pointing to <a href="http://localhost:3001">http://localhost:3001</a>

<h3>Production</h3>

Run the following command:<br />

npm run build<br />
